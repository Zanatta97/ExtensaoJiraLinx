## Português

### Sobre o Projeto
Uma extensão simples do Chrome para uso interno na Linx, com o objetivo de facilitar o dia a dia dos meus colegas de trabalho. Utilizado **JavaScript** para formar a URL com o número da ISSUE ou o texto de pesquisa para facilitar a busca por issues dentro do Jira. Após isto, é realizado o login automaticamente injetando scripts na página de login do Jira caso necessário.

Esta extensão se tornou necessária, pois para o usuário geral do suporte, as opções de pesquisa foram removidas, tornando mais difícil a busca por ISSUES dentro do Jira.

## English

### About the Project
A simple Chrome extension built for internal use at Linx, designed to streamline the daily workflow of my colleagues. It uses **JavaScript** to construct URLs based on issue numbers or search terms, making Jira searches faster. Additionally, it handles automatic login by injecting scripts into the Jira login page when necessary.

This extension became essential after search options were restricted for general support users, which had made finding issues within Jira more difficult.
